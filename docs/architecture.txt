---------------------------------------------------------------------------
REQUEST FLOW (ANALYTICS EXAMPLE)
---------------------------------------------------------------------------

1. User opens Analytics page (Frontend)
2. Frontend calls API via api.js
3. Request hits backend/app.py
4. Auth validated via auth.py
5. Backend queries database.py
6. Analytics computed (detections, recordings, etc.)
7. Response sent to frontend
8. UI renders charts (Analytics.jsx)
